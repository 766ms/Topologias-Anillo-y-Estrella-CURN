# Diseño e implementación de un recurso educativo de las diferentes topologías de redes:
Este proyecto presenta el desarrollo de un recurso pedagógico físico diseñado para la enseñanza práctica de arquitecturas de red. Utilizando la plataforma de hardware abierto Arduino UNO, se han implementado prototipos funcionales que permiten visualizar el comportamiento dinámico de los datos en entornos de red reales.
El recurso educativo facilita la comprensión de conceptos abstractos mediante la interacción directa con componentes electrónicos, permitiendo observar la dependencia del nodo central en configuraciones de estrella y la propagación de señales en estructuras de anillo.
# Objetivos del Recurso:
- Visualización Tangible: Representar el flujo de información mediante una secuencia de indicadores LED.
- Experimentación de Campo: Permitir al estudiante interactuar con el hardware (pulsadores y nodos) para simular eventos de red.
- Validación de Protocolos: Demostrar la lógica de control necesaria para gestionar diferentes jerarquías de conexión.
# Especificaciones Técnicas
Unidad de Control: Microcontrolador Arduino UNO R3.
Infraestructura: Placa de pruebas (Protoboard) de 3260 puntos para una organización clara del cableado.
Componentes de Red: LEDs de alta eficiencia (Nodos), Resistencias de protección y Pulsadores (Disparadores de eventos).
Lenguaje de Programación: C++ (entorno Arduino IDE).
